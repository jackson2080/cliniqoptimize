# CliniqOptimize v0.4

**AI-powered Lean Six Sigma DMAIC platform for healthcare quality improvement.**

A single-file, browser-based clinical operations analytics tool built for healthcare quality teams. No installation. No server. Open `index.html` and go.

---

## Features

- **Live Sigma Dashboard** — Real-time σ level, DPMO, and COPQ calculated from your entered metrics across 9 departments (ED, ICU, Pharmacy, OR, Lab, Radiology, Ambulatory, Inpatient, Urgent Care)
- **Full DMAIC Workflow** — AI-assisted Define (project charter), Measure (process capability), Analyze (root cause), Improve (solutions), Control (monitoring plan)
- **Quality Tools** — FMEA, Value Stream Mapping, AI Suggestions engine, Human Factors analysis
- **Belt-Differentiated Output** — Yellow / Green / Black Belt formatting across all AI reports
- **EHR-Aware Analysis** — Epic, Cerner, Meditech, Allscripts, athenahealth, eClinicalWorks context in every prompt
- **124-Citation Evidence Library** — DOIs and PMIDs embedded in root cause analysis
- **COPQ Calculator** — Cost of Poor Quality estimated per department and metric
- **Printable Reports** — Charter, analysis, and control plan print views

## Usage

1. Open `index.html` in any modern browser
2. Complete onboarding (select EHR, belt level, add Anthropic API key)
3. Enter your department's real metric values on the Dashboard
4. Navigate DMAIC tabs to generate AI-assisted reports

## Requirements

- Modern browser (Chrome, Edge, Firefox, Safari)
- Anthropic API key (Claude Sonnet) for AI-generated reports
- No installation, no dependencies, no server

## AI Disclaimer

All AI-generated reports, citations, benchmarks, and recommendations require verification before clinical or operational use. This tool is for quality improvement education and project planning only — not a substitute for clinical judgment or validated institutional data.
